# Video Game Sales Prediction

## Goal
This project aims to predict global video game sales using machine learning models.

## Dataset
The dataset contains information about video games such as platform, genre, publisher, year of release, and sales figures.

## Project Structure
- data: raw dataset files
- notebooks: EDA and experiments
- src: main ML scripts
- outputs: trained models and predictions

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib
